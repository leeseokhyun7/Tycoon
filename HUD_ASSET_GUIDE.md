# HUD Asset Guide

본게임 상단 HUD는 아래 PNG 파일을 넣으면 자동으로 스킨이 적용됩니다.

파일 위치는 모두 `assets/new/` 입니다.

## 권장 PNG 사이즈

- `hud-title.png`: `500 x 128 px`
  - 상단 왼쪽 `붕어빵거리 / DAY` 타이틀 영역
  - 실제 표시 크기 약 `250 x 64 px`

- `hud-sales.png`: `454 x 84 px`
  - `매출` 카드 배경
  - 실제 표시 크기 약 `227 x 42 px`

- `hud-time.png`: `354 x 84 px`
  - `시간` 카드 배경
  - 실제 표시 크기 약 `177 x 42 px`

- `hud-goal.png`: `820 x 96 px`
  - `목표` 진행바 전체 배경
  - 실제 표시 크기 약 `410 x 48 px`

## 제작 규칙

- 투명 PNG로 만들면 기존 숫자, 시간, 진행바 텍스트가 위에 그대로 보입니다.
- 글자까지 이미지에 넣고 싶으면 코드의 실제 텍스트와 겹칠 수 있습니다.
- 숫자와 시간은 게임 중 계속 바뀌므로 이미지에는 배경 프레임만 넣는 것을 권장합니다.
- 파일명만 정확히 맞추면 코드 수정 없이 바로 교체됩니다.
