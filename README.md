# 붕어빵 거리: 스노우마켓

첨부 참고 이미지의 모바일 캐주얼 타이쿤 감성을 바탕으로 다시 만든 붕어빵 가게 운영 게임입니다. 원작 그래픽과 캐릭터를 복제하지 않고, 겨울 거리와 귀여운 손님, 가판대, 조리판 중심의 독자적인 웹 게임으로 구성했습니다.

## 실행

`index.html`을 브라우저로 열면 바로 실행됩니다. 설치 과정은 없습니다.

## 주요 변경

- 세로형 모바일 게임 화면으로 UI 전면 재설계
- 겨울 거리 배경, 가판대, 마스코트, 손님 캐릭터 SVG 자산 추가
- 손님 주문에 수량, 대기시간, 급한 주문, 보석 보상 적용
- 반죽, 속재료, 뒤집기, 서빙 단계의 조리 타이밍 시스템 개편
- 완벽 조리 보너스, 콤보 배율, 목표 매출, 영업 종료 보너스 추가
- 상점에서 붕어틀, 열판, 포장, 간판, 진열대 업그레이드 가능
- 치즈, 고구마, 말차 레시피 해금 시스템 추가
- 일일 미션과 브라우저 저장 진행도 추가

## 파일

- `index.html`: 게임 화면 구조
- `styles.css`: 모바일 게임 UI, 캐릭터, 조리판, 상점 스타일
- `script.js`: 주문, 조리, 상점, 미션, 저장 로직
- `assets/`: 게임용 SVG 그래픽 자산
