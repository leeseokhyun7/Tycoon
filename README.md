# AGENTS.md
모바일 붕어빵 타이쿤 게임을 더 완성도 있게 개선해줘.
AGENTS.md 기준을 따르고, 단순 데모가 아니라 실제로 플레이 가능한 상태로 만들어줘.
작업 후 모바일 UI 겹침, 콘솔 오류, 시작/진행/종료/다시하기 흐름을 검증해줘.

## 기본 목표
이 저장소는 모바일 웹 미니게임 모음입니다.
코드는 단순히 작동하는 수준이 아니라, 사용자가 바로 플레이할 수 있는 완성도까지 갖춰야 합니다.

## 공통 기준
- 모바일 우선으로 구현한다.
- 390px 너비 화면에서 버튼, UI, 텍스트가 겹치지 않아야 한다.
- 게임 시작, 진행, 실패, 성공, 다시하기 흐름이 있어야 한다.
- 조작 버튼은 엄지로 누르기 쉬운 위치에 둔다.
- 화면 중앙을 불필요한 텍스트나 버튼으로 가리지 않는다.
- 설명 텍스트는 짧게 하고, 플레이를 방해하지 않게 한다.
- CSS는 임시 느낌이 나지 않게 정리한다.
- 색상, 간격, 버튼 크기, 폰트 크기는 게임 분위기에 맞게 조정한다.

## 게임 완성 기준
작업이 끝나면 아래를 반드시 확인한다.

1. 브라우저 콘솔 오류 없음
2. 모바일 화면에서 UI 겹침 없음
3. 게임 시작 가능
4. 게임 진행 가능
5. 실패 또는 종료 조건 작동
6. 다시하기 가능
7. 수정한 파일 목록 요약
8. 검증한 항목과 검증하지 못한 항목 구분

## 작업 방식
- 기존 파일 구조를 먼저 파악한다.
- 불필요하게 파일을 많이 늘리지 않는다.
- 단순한 데모가 아니라 실제로 플레이 가능한 결과물을 만든다.
- 애매한 부분은 가장 자연스러운 게임 UX 기준으로 판단한다.
- 작업 후에는 직접 실행하거나 가능한 테스트 명령을 실행한다.

# 붕어빵 거리: 스노우마켓

첨부 참고 이미지의 모바일 캐주얼 타이쿤 감성을 바탕으로 다시 만든 붕어빵 가게 운영 게임입니다. 원작 그래픽과 캐릭터를 복제하지 않고, 겨울 거리와 귀여운 손님, 가판대, 조리판 중심의 독자적인 웹 게임으로 구성했습니다.

## 실행

`index.html`을 브라우저로 열면 바로 실행됩니다. 설치 과정은 없습니다.

## 주요 변경

- 사용자가 제공한 `assets/new` 이미지 세트를 게임 화면에 적용
- 세로형 모바일 게임 화면으로 UI 전면 재설계
- 겨울 거리 배경, 가판대, 마스코트, 손님 캐릭터 SVG 자산 추가
- 손님 주문에 수량, 대기시간, 급한 주문, 보석 보상 적용
- 초반에는 1개 주문, 이후 2~3개 주문과 여러 맛 혼합 주문 등장
- 손님 말풍선에 주문 대사와 메뉴별 수량 표시
- 반죽, 속재료, 뒤집기, 서빙 단계의 조리 타이밍 시스템 개편
- 완벽 조리 보너스, 콤보 배율, 목표 매출, 영업 종료 보너스 추가
- 상점에서 붕어틀, 열판, 포장, 간판, 진열대 업그레이드 가능
- 치즈, 고구마, 말차 레시피 해금 시스템 추가
- 일일 미션과 브라우저 저장 진행도 추가

## 파일

- `index.html`: 게임 화면 구조
- `styles.css`: 모바일 게임 UI, 캐릭터, 조리판, 상점 스타일
- `script.js`: 주문, 조리, 상점, 미션, 저장 로직
- `assets/new/customer.png`: 손님 캐릭터 스프라이트
- `assets/new/bbang.png`: 붕어빵 스프라이트
- `assets/new/world.png`: 배경 이미지
- `assets/new/ui.png`: 가판대와 UI 질감 이미지
- `assets/generated/`: 체크무늬 배경을 제거한 실제 게임용 투명 컷아웃
